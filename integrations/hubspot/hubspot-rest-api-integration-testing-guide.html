    應用程式介面整合測試指南

    在本主題中，您將學習如何使用 REST API 驗證視訊雲端與 HubSpot 之間的整合。

    使用 REST API 設定新的 HubSpot 整合時，測試並確認視訊檢視資料確實已傳遞至 HubSpot 可能會很有挑戰性。在本主題中，您將學習如何系統地測試和確認整合是否正常運作，以及檢視資料和潛在客戶正在傳送到 HubSpot。

    注意：本主題適用於驗證使用 REST API 所建立的中樞連線。
    注意：有關如何配置Audience模塊以與HubSpot一起使用的信息，請參閱 使用REST API將視頻雲與HubSpot集成。

    開始之前

    驗證整合之前，假設您有：

    • 成功完成Brightcove Audience與HubSpot集成
    • 創建了至少一個啟用受眾群體的播放器並將其配置為使用HubSpot連接

    配置測試線索表格並發布視頻

    為了進行測試，您將需要在Audience中創建和配置自定義潛在客戶表單。這是一個 HubSpot 表單，其內嵌程式碼已貼到受眾中。即使您不會在生產環境中使用潛在客戶表單，也可以使用該潛在客戶表單來確保您在數據庫和頁面中均已知曉。若要建立 HubSpot 表單並設定對象使用它，請依照下列步驟執行。

    1. 登入您的帳戶。
    2. 建立和設定 HubSpot 表單的樣式。有關創建HubSpot表單的信息，請參見為HubSpot創建自定義潛在客戶表單。
    3. 請點擊分享在頁面頂部。
    4. 將表單嵌入代碼複製到剪貼板。
    5. 返回「觀眾」模組。
    6. 使用 HubSpot 表單內嵌程式碼建立新的潛在客戶表單。
    7. 設置您的自定義窗體以在5秒標記處顯示該窗體。這樣，您將確切知道何時顯示。
    8. 勾選此方塊，即使該潛在客戶已知，也一律顯示潛在客戶表單。這將確保在測試過程中填寫表格後，表格繼續顯示。
    9. 保存表格。
    10. 使用 HubSpot 連線設定 Brightcove 播放程式，以使用剛建立的 HubSpot 潛在客戶表單。

    生成預覽鏈接以觀看視頻

    要生成視頻的預覽鏈接，請按照下列步驟操作。

    1. 打開媒體模塊。
    2. 選擇一個視頻並將其發佈到網絡播放器。確保使用潛在客戶表單選擇支持受眾的播放器。
    3. 按一下預覽連結，在新的瀏覽器視窗中開啟視訊。通過這種方式進行測試，可以消除嵌入HTML頁面的播放器可能引入的任何潛在變量。
    4. 確認播放了視頻，並且線索表顯示在5秒標記處。
    5. 使用唯一的聯絡資訊和電子郵件地址填寫潛在客戶表單 (例如名人姓名或包含「bctest」字樣的名稱)，以便稍後在 HubSpot 中輕鬆找到該表單。這消除了任何瀏覽器，播放器或數據庫級別的變量。透過以新使用者身分填寫表單，我們可以確定您現在是 HubSpot 中的已知使用者。

    確認受眾已成功擷取資料

    在 HubSpot 中尋找視訊觀看資料之前，請先確認觀眾確實擷取了視訊檢視畫面。請注意，觀看數據最多可能需要3個小時才能顯示在受眾群體用戶界面中。要確認受眾群體已成功捕獲數據，請按照下列步驟操作。

    1. 打開聽眾模塊。
    2. 按一下左側導覽中的「所有最近的活動」。
    3. 確認有一個帶有最近時間戳記的事件，該事件顯示了剛剛觀看的視頻的名稱。事件狀態應為未處理。這表示觀眾已擷取檢視，但尚未嘗試將其同步至 HubSpot。
    4. 要手動觸發受眾群體和HubSpot之間的同步，請點擊同步活動在左側導航欄中。
    5. 來自選擇連接下拉列表中，選擇您的HubSpot集成。一種測試同步按鈕應該出現。
    6. 按一下 [ 測試同步 ] 以觸發手動同步。請注意，您限於觸發手動同步的頻率。根據預設，觀眾會每小時將檢視同步至 HubSpot 一次。
    7. 按一下左側導覽中的「所有最近的活動」。
    8. 您的觀看事件狀態為排隊。資料可能需要 15 分鐘左右的時間才能實際同步到 HubSpot。數據同步後，狀態將更改為以下兩種狀態之一：
      • 已同步 -表示該用戶是已知用戶，並且數據已成功傳送到其活動日誌。數據也應該在幾分鐘內可以在 HubSpot 中查看。
      • unknown_contact -表示該用戶被視為未知用戶，並且HubSpot無法接受數據。如果你得到一個unknown_contact狀態，請檢查您是否使用正確的播放器發布了視頻，是否有HubSpot表單以及是否正在使用預覽鏈接（而不是嵌入到頁面中）進行測試。

    確認視訊檢視資料位於 HubSpot

    受眾群體顯示後，已同步狀態，請按照以下步驟驗證數據是否在HubSpot中。

    注意：HubSpot UI 需要 5-10 分鐘才能使用從受眾接收到的新資料進行更新。
    1. 登入您的帳戶。
    2. 在HubSpot中導航到聯絡人區域並找到您填寫潛在客戶表單時使用的唯一名稱或電子郵件地址。
    3. 按一下連絡人以開啟活動記錄。
    4. 點擊篩選活動下拉菜單（活動日誌上方的右側）並確認Brightcove Video Connect被檢查。這將確保視頻觀看活動顯示在活動日誌中。
    5. 在活動日誌中，查找說明User_Name發生了Brightcove Video Connect事件。點擊顯示詳細資料查看視頻觀看事件的數據。

    HubSpot 要求透過其 API 傳遞資料，才能將資料指派給已知的連絡人。HubSpot 只接受已知聯絡人的資料。未知聯繫人的數據將不會同步。如果您不使用潛在客戶表單，則未知用戶將無法轉換為已知用戶，因此不會同步任何數據。
