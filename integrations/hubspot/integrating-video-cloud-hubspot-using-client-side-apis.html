    使用用戶端 API 整合視訊雲端與 HubSpot

    本主題說明如何使用客戶端API將Audience模塊配置為與HubSpot一起使用。

    觀眾模組可讓視訊雲端同步觀看資料從布萊特灣播放器到 HubSpot。一旦進入 HubSpot，此資料可用於細分、行銷活動邏輯、報告、潛在客戶評分，以及個人化通訊。

    有兩個選項可以整合視訊雲端與 HubSpot：

    • 使用標準 (REST API) 方法 -使用此方法，受眾只會追蹤已知檢視者的檢視資料。根據預設，潛在客戶表單只會顯示給未知的觀眾。有關此類集成的信息，請參見使用REST API將視頻雲與HubSpot集成。
    • 使用高容量（客戶端 API）方法 -使用此方法，Audience 將追踪所有觀眾的觀看數據。此方法將在本主題中說明。

    與 HubSpot 的觀眾模組整合包含兩個主要元件：

    • 可追蹤影片觀看資料的 Brightcove 播放程式外掛程式
    • 將追蹤資料從視訊雲端傳送至 HubSpot 的同步處理程序

    需求

    若要搭配 HubSpot 使用對象模組，您必須：

    • 擁有一個帳戶
    • 擁有專業或企業視訊雲端帳戶
    • 在開啟 Audience 模組時出現的初始啟動畫面中提交必要資訊，讓帳戶啟用 Audience 模組
    • 第一次使用模組時，按一下服務合約
    • 發布您的播放器時使用高級嵌入（頁面內）發布代碼; 標準嵌入（iframe）發布代碼將無法工作，因為插件將在 iframe 內運行，因此將無法訪問任何父頁面 URL 參數，並且可能無法獲得正確的 cookie

    功能

    視訊雲端-HubSpot 整合的一些主要功能包括：

    • 潛在客戶表單 -建立 HubSpot 表單，這些表單會在觀眾觀看影片時自動顯示。如需詳細資訊，請參閱為 HubSpot 建立自訂商機表單。
    • 資料格式 -檢視資料會以自訂的時間表事件類型傳送。複寫到 HubSpot 的資料如下所示：
      • 視訊名稱
      • 視訊 ID
      • 帳號 ID (視訊雲端)
      • 網頁網址 (參照頁面的網址)
      • 玩家 ID（視頻雲端播放器）
      • 百分比 (25%, 50%, 75%, 95%)
    • 資料傳輸 -使用自動化程序將檢視資料傳送至 HubSpot；不過，HubSpot 平台中的某些報告可能需要一些時間才能重新整理
    • 資料保留 -收集的資料可選擇性地保留在 Video Cloud 中長達 6 個月

    設定連線

    請依照下列步驟設定視訊雲端和 HubSpot 之間的連線。

    建立視訊雲端與 HubSpot 之間的連線

    1. 開啟對象模組，然後按一下左側導覽中的資料連線。
    2. 按一下「新增連線」。
    3. 選取「中樞點」作為整合。
      注意：可用的整合是根據您的 Video Cloud 訂閱而定。
    4. (選擇性) 輸入「連線名稱」。
    5. 按一下「連線」。
    6. 選擇高容量（客戶端API）。
    7. 按一下「儲存」。

    現在已建立到HubSpot的視頻雲連接。

    設定整合

    建立視頻雲到HubSpot的連接後，請按照以下步驟配置集成。

    1. 集成需要您的HubSpot帳戶ID。點擊空的鏈接，輸入您的HubSpot帳戶ID，然後單擊保存。
      注意：不支持具有相同帳戶ID的多個HubSpot連接。
    2. 依預設， [連線狀態] 將會是 [ 已啟用 ]。連線狀態會決定視訊雲端視訊參與資料是否傳送至 HubSpot。
      • 已啟用 -資料會收集並傳送至 HubSpot
      • 停用 -未收集任何資料 (已停用整合)

    若要移除與 HubSpot 的整合，請按一下 [ 中斷連線 ]。Audience 模組支援多個行銷平台的連線。按一下 [ 新增連線 ] 以設定其他整合。

    建立具有受眾功能的播放器

    視訊參與資料只會傳送至 HubSpot，當您使用支援聽眾的視訊播放器時。如需建立啟用受眾的 Brightcove 播放器的詳細資訊，請參閱建立啟用受眾的播放程式。

    傳遞追蹤 ID

    對象模組支援將追蹤 ID 值傳遞給 HubSpot 的能力。例如，追蹤 ID 可能用來追蹤廣告活動 ID。追蹤 ID 可以附加至顯示已啟用受眾播放程式的 URL，或新增至播放程式內嵌程式碼。

    • 範例：附加至網址
      http://www.myurl.com/video.html? 按下滑鼠右鍵 = 12345
    • 範例：新增至玩家嵌入程式碼
      <視頻數據視頻代碼 = 
         資料帳戶 = "1486906377」 
         數據播放器 =「默認」 
         數據嵌入 =「默認」 
         資料應用程式識別碼 
         類 =「視頻 JS」
         資料直接追蹤器 = "12345"  
         控> < 制/視訊 >
  <指令碼的最新版 > < 本 >

    接著，觀眾會將值傳遞給 HubSpot 做為同步的另一個欄位。

    識別檢視者

    對於 Hub S pot 整合，該外掛程式會將 HubSpot 的 JavaScript 追蹤程式碼放在使用者的瀏覽器上。此 Cookie 用於追蹤 HubSpot 中的訪客身分，並將布萊特灣查看資料與聯絡人建立關聯。如果使用者在播放程式載入時已經有 hubspotutk Cookie，觀眾將向 HubSpot 提出 API 請求，並詢問 Cookie 值是否對應於已知的連絡人，如果是這樣，觀眾會將檢視資料與該使用者建立關聯。
