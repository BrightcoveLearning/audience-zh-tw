    使用對象模組建立自訂整合

    本主題說明瞭如何配置Audience模塊以進行自定義集成。

    受眾群體模塊允許Video Cloud同步啟用了受眾群體的觀看數據  Brightcove播放器可用於流行的營銷自動化平台。受眾模塊也可以配置為自定義集成。通過自定義集成，查看數據將本地保存在Video Cloud中，並且Audience API用於以編程方式提取數據。

    Audience模塊集成由兩個主要組件組成：

    • 可追蹤影片觀看資料的 Brightcove 播放程式外掛程式
    • 使用Audience API檢索觀看數據的自定義開發

    需求

    要使用受眾模塊，您必須：

    • 擁有專業或企業視訊雲端帳戶
    • 在開啟 Audience 模組時出現的初始啟動畫面中提交必要資訊，讓帳戶啟用 Audience 模組
    • 第一次使用模組時，按一下服務合約
    • 為了收集潛在客戶，潛在客戶表單必須配置並分配給玩家
    • 發布播放器時使用高級（頁內）嵌入代碼；標準（iframe）嵌入代碼將不起作用，因為該插件將在iframe中運行，因此將無法訪問父頁面的網址參數

    設定連線

    請按照以下步驟配置自定義集成。

    1. 開啟對象模組，然後按一下左側導覽中的資料連線。
    2. 按一下「新增連線」。
    3. 選擇自訂作為整合。
      注意：可用的整合是根據您的 Video Cloud 訂閱而定。
    4. (選擇性) 輸入「連線名稱」。
    5. 按一下「連線」。

    現在，自定義連接已建立並可以使用。

    設定整合

    建立連接後，您可以配置集成。

    1. 依預設， [連線狀態] 將會是 [ 已啟用 ]。的連接狀態確定是否保存了Video Cloud視頻參與度數據。
      • 已啟用 -收集數據並將其保存
      • 停用 -未收集任何資料 (已停用整合)
    2. 選取資料保留原則。資料保留原則決定檢視資料和潛在客戶表單資料 (如果使用 Audience 潛在客戶表單) 將保留在您的 Video Cloud 帳戶中的時間長度。
      注意：如果選擇沒有保留，將不會保存任何數據。使用API下載數據時，將不會有任何內容。

    建立具有受眾功能的播放器

    視頻參與度數據僅在以下情況下保存 您正在使用支持觀眾的視頻播放器。如需建立啟用受眾的 Brightcove 播放器的詳細資訊，請參閱建立啟用受眾的播放程式。

    傳遞追蹤 ID

    受眾模塊支持傳遞跟踪ID的功能。例如，追蹤 ID 可能用來追蹤廣告活動 ID。追蹤 ID 可以附加至顯示已啟用受眾播放程式的 URL，或新增至播放程式內嵌程式碼。

    • 範例：附加至網址
      http://www.myurl.com/video.html? 按下滑鼠右鍵 = 12345
    • 範例：新增至玩家嵌入程式碼
    <video data-video-id =“ 6033402539001”
        資料帳戶 = "1486906377」
        數據播放器 =「默認」
        數據嵌入 =「默認」
        資料應用程式識別碼
        類 =「視頻 JS」
        資料直接追蹤器 = "12345" 
        控> < 制/視訊 >
<指令碼的最新版 > < 本 >

    識別檢視者

    使用自定義集成時，您有責任識別查看者。Audience插件需要兩個關鍵信息：

    描述 網址參數 嵌入參數 價格
    觀眾是否已知（需要潛在客戶形式） bcknownuser data-bc-known-user 0或1
    什麼是 GUID (檢視事件追蹤所需) bccustomguid data-bc-custom-guid string
     

    這些值可以通過URL參數或播放器嵌入代碼傳遞。若要進行檢視者追蹤，使用者識別碼應當做顯示視訊的頁面上的 URL 參數傳遞。潛在客戶表單需要已知的用戶參數（如果您希望潛在客戶查看者有條件地顯示潛在客戶表單）。

    注意：為了使播放器/插件讀取URL參數，應使用高級（頁內）嵌入代碼。標準（iframe）嵌入代碼將無法工作，因為插件將在 iframe 內部運行，因此無法訪問具有該參數的父頁面 URL。

    檢視視訊資料

    使用自定義集成時，有幾個選項可用於查看視頻數據：

    • 檢視器活動 (事件) -作為「觀眾」模組的一部分，此選項可用來檢視「觀眾」模組內的視訊資料。如需資訊，請參閱檢視對象模組中的播放器事件。
    • 匯出資料 -作為對象模組的一部分提供，此選項可用來匯出視訊檢視和潛在客戶資料。如需資訊，請參閱從對象模組匯出受眾資料。
    • 受眾 API -受眾 API 可用於以程式設計方式擷取視訊檢視和潛在客戶資料。如需詳細資訊，請參閱概述：受眾 API 。
