    建立自訂潛在客戶表單

    在本主題中，您將學習如何使用 Salesforce 表單擷取布萊特灣玩家內的潛在客戶資料。

    觀眾模組提供建立自訂潛在客戶表單的功能，讓您可以在 Brightcove 播放器內播放影片時擷取觀眾資訊。潛在客戶資料會傳送給 Salesforce，並由 Salesforce 處理。

    若要建立自訂商機表單，必須執行下列步驟。

    1. 在Salesforce中創建表單
    2. 複製和編輯HTML表單
    3. 在對象模組中建立自訂潛在客群表單
    4. 將潛在客戶表單指派給玩家
    5. 驗證Salesforce中的表單數據
    注意：如需標準潛在客戶表單的資訊，可擷取檢視者資訊並將資料儲存在 Video Cloud 中的資訊，請參閱建立受眾潛在客戶表單。

    建立表單

    Salesforce 中的「網路對潛在客戶」功能可用來產生網站訪客的潛在客戶資訊。這些形式也可以顯示在布萊特灣玩家的內部。下面的步驟將創建一個簡單的表單來捕獲名字，姓氏，電子郵件地址，公司，城市和州/省。如需網路對商機功能的完整詳細資訊，請參閱 S alesforce 文件。

    1. 登入您的帳戶。
    2. 從設定，輸入Web-to-Lead在快速查找框中，然後單擊網絡對領導。
    3. 按一下 [ 建立網路對商機表單 ]。
    4. 選取要包含在網路對商機表單上的欄位。使用「添加」和「刪除」箭頭在「可用字段」列表和「所選字段」列表之間移動字段，并使用向上和向下箭頭來更改字段在表單中的順序。
    5. 輸入「傳回網址」。在此範例中，傳回 URL 將設定為 http://www.brightcove.com。當表單用作 Audience 模組內的自訂潛在客戶表單時，將忽略此值。
    6. 按一下「產生」。表單 HTML 將被顯示。

    複製和編輯表單 HTML

    若要使用 Audience 模組內的表單，您必須複製表單 HTML。此外，您可能想要對表單做一些基本的樣式。在下面的步驟中，我們將表單 HTML 複製到 HTML 編輯器，然後對表單進行一些樣式更改。

    默認情況下，當表單顯示在 Brightcove 播放器內時，表單將顯示在玩家的透明背景上。使用 CSS 來設置背景顏色。

    1. 將 HTML 複製到剪貼簿。
    2. 按一下「完成」。
    3. 將表單 HTML 貼到您選擇的 HTML 編輯器中。您可能想要使用編輯器的程式碼格式化功能來格式化程式碼，以便更容易閱讀。
    4. 使用 HTML 編輯器，添加以下 CSS 將背景顏色更改為灰色。
      <style>
      body {
      background-color: rgba(204,204,204,0.9);
      margin: 0;
      padding: 20px;
      height: 100%;
      }
      </style>
    5. （可選）要確定受眾群體產生了哪些潛在客戶，可以在<form>標籤：
      <input type="hidden" name="lead_source" value="Video Cloud Audience">
    6. 將 HTML 複製到剪貼簿。

    在對象模組中建立自訂潛在客群表單

    在 Salesforce 中產生表單 HTML 後，請在 Audience 模組中建立自訂潛在客戶表單，並將潛在客戶表單與啟用受眾的播放器建立關聯。

    1. 打開聽眾模塊。
    2. 按一下左側導覽中的「潛在客戶表單」。如果已建立任何潛在客戶表單，則會顯示這些表單。
    3. 按一下「建立商機塑形」。
    4. 為潛在客戶表單指定描述性的「名稱」。
    5. 選取表單語言。
    6. 選取「表單計時」。表單時間會決定何時顯示潛在客戶表單。
    7. 將表單類型設定為 [ 自訂商機表單 ]。
    8. 粘貼在表格HTML來自Salesforce。
    9. ( 選擇性) 貼上確認 HTML 。確認視圖將顯示 5 秒鐘，然後視頻將繼續播放。如果沒有包含確認 HTML，視頻將在表單提交後繼續播放。
    10. ( 選擇性) 按一下「預覽表單」以開啟表單的預覽。
    11. 按一下 [ 儲存] 儲存商機表單。

    將潛在客戶表單指派給玩家

    一旦建立了潛在客戶表單，就需要將其指派給玩家。請依照下列步驟將潛在客戶表單指派給玩家。

    1. 按一下左側導覽中的「觀眾播放器」。
    2. 找到您的玩家並使用潛在客戶表單下拉式清單來選取潛在客戶表單。

    現在已完成設定，您已準備好擷取潛在客戶資訊。使用受眾播放器發佈影片時，會根據表單時間設定顯示主題表單。

    驗證表單資料

    使用自訂潛在客戶表單擷取的潛在客戶，會新增為您的 Salesforce 帳戶中的潛在客戶。
