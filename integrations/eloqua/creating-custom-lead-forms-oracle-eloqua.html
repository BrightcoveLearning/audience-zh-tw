    建立 Oracle Eloqua 的自訂潛在客戶表單

    在本主題中，您將學習如何使用Oracle Eloqua表單在Brightcove Player內部捕獲潛在客戶數據。

    觀眾模組提供建立潛在客戶表單的能力，讓您在 Brightcove 玩家內部播放影片時，擷取觀眾資訊。這使您能夠包含您需要的任何字段（可見或隱藏），並且讓您能夠利用漸進式分析（如果可用）等功能。商機資料會傳送至 Eloqua，並由 Eloqua 處理。

    若要建立自訂商機表單，必須執行下列步驟。

    1. 在Eloqua中創建一個表單
    2. 複製表單HTML
    3. 設置表格樣式
    4. 在對象模組中建立自訂潛在客群表單
    5. 將潛在客戶表單指派給玩家
    6. 在Eloqua中驗證表單數據
    注意：如需標準潛在客戶表單的資訊，可擷取檢視者資訊並將資料儲存在 Video Cloud 中的資訊，請參閱建立受眾潛在客戶表單。

    在 Eloqua 中創建表單

    Eloqua提供了創建表單來捕獲查看者信息的功能。這些表格可以顯示在Brightcove Players內部。擷取的資料可用於自訂未來的通訊。下面的步驟將創建一個簡單的表單來捕獲名字，姓氏和電子郵件地址。有關 Eloqua 表單的完整詳細信息，請參閱 Eloqua 文檔。

    1. 登入您的 Eloqua 帳戶。
    2. 切換作業選項至資產 > 表單，然後按一下「建立表單」。表單範本選擇器隨即開啟。
    3. 按一下空白表單，然後按一下 [ 選擇 ]。
    4. 在左側導覽中，展開「欄位群組」面板。
    5. 在「欄位群組」面板中，按一下「基本聯絡資訊」，然後將其拖曳至表單範本。
    6. 確認欄位已新增至表單。
    7. 依預設，會使用欄位合併來新增欄位。在“受眾群體”中使用時，這會導致奇怪的文本出現在字段中。若要移除欄位合併，請按一下「名字」欄位來選取該欄位。
    8. 在左側導覽面板中，找到「一般屬性」區段，並將「預先填入欄位資料」屬性設定為「無」。
    9. 對 [ 姓氏 ] 和 [ 電子郵件地址 ] 欄位重複此程序。
    10. 按一下「儲存」。
    11. 為表單命名，然後按一下 [ 儲存 ]。

    複製表單 HTML

    若要使用 Audience 模組內的表單，您必須複製表單 HTML。請依照下列步驟複製表單 HTML。

    1. 按一下「動作」> 檢視表單 HTML...表單 HTML 將被顯示。
    2. 將整個 HTML 複製到剪貼簿。按一下「關閉」。
    3. 將表單 HTML 貼到您選擇的 HTML 編輯器中。您可能想要使用編輯器的程式碼格式化功能來格式化程式碼，以便更容易閱讀。

    設定表單樣式

    默認情況下，當表單顯示在 Brightcove 播放器內時，表單將顯示在玩家上的透明背景上。您可能會想使用CSS對錶單做一些基本的樣式化。以下各節提供兩個範例，說明如何設定表單樣式。也會提供相關聯的 CSS。您可能需要調整 CSS 以符合您的標準顏色或處理其他字段的樣式。

    簡單表單樣式範例

    這個例子使用 CSS 將表單的背景顏色更改為灰色和表單將出現如下：

    對於此樣式，請將以下 CSS 添加到表單 HTML style標記的頂部。

    主體 {
      背景顏色：重量（204,204,0.9）;
      邊界：0；
      填充：20 像素；
      高度：100%；
}

    在對象模組中建立自訂潛在客群表單

    在Eloqua中創建表單HTML並根據需要設置樣式後，請在“受眾群體”模塊中創建自定義潛在客戶表單，然後將潛在客戶表單與啟用了受眾群體的播放器關聯。

    1. 打開聽眾模塊。
    2. 按一下左側導覽中的「潛在客戶表單」。如果已建立任何潛在客戶表單，則會顯示這些表單。
    3. 按一下「建立商機塑形」。
    4. 為潛在客戶表單指定描述性的「名稱」。
    5. 選取表單語言。
    6. 選取「表單計時」。表單時間會決定何時顯示潛在客戶表單。
    7. 將表單類型設定為 [ 自訂商機表單 ]。
    8. 粘貼在表格HTML。
    9. ( 選擇性) 貼上確認 HTML 。確認視圖將顯示 5 秒鐘，然後視頻將繼續播放。如果沒有包含確認 HTML，視頻將在表單提交後繼續播放。
    10. ( 選擇性) 按一下「預覽表單」以開啟表單的預覽。
    11. 按一下 [ 儲存] 儲存商機表單。

    將潛在客戶表單指派給玩家

    一旦建立了潛在客戶表單，就需要將其指派給玩家。請依照下列步驟將潛在客戶表單指派給玩家。

    1. 按一下左側導覽中的「觀眾播放器」。
    2. 找到您的玩家並使用潛在客戶表單下拉式清單來選取潛在客戶表單。

    現在已完成設定，您已準備好擷取潛在客戶資訊。使用受眾播放器發佈影片時，會根據表單時間設定顯示主題表單。

    驗證 Eloqua 中的表單數據

    有幾種方法可以驗證表單資料是否已傳遞給 Eloqua。請依照下列步驟來檢視 Eloqua 中的表單資料。

    1. 登入您的 Eloqua 帳戶。
    2. 切換作業選項至資產 > 表單。
    3. 在左側導覽中，按一下「所有檔案」連結以顯示所有表單。
    4. 按兩下表單名稱以開啟它。
    5. 按一下「動作」> 檢視提交資料...表單數據將被顯示。
